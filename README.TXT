WEB CONTENT OPTION


Questo disco contiene un visualizzatore WEB delle immagini.

Esso parte automaticamente sui sistemi Microsoft Windows. 

Altrimenti cliccare su risorse del computer e poi su CDStart, scegliere il visualizzatore

Per attivare il visualizzatore DICOM fare un click sulla scritta :"Visualizzatore Professionale DICOM"

Per attivare il visualizzatore WEB fare un click sulla scritta :"visualizzatore Standard WEB"

Non rimuovere il disco dal lettore se il visualizzatore è ancora attivo.
